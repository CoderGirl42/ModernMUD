Editing Exits

Back to the index

Exits are used when traveling from one room to another. They can be open passages or doorways.

The edit exit dialog can be reached by clicking on any of the directional indicators (n, s, e, w, nw, sw, ne, se, up and dn) within the room edit dialog.

Target Room: This is the room the exit leads to.

You can click the "Find" button to select the target room from a list, or click "New" to create a new room and link it to this exit. In this case, clicking "New" would link the east exit in room 6709 to a new, empty room and link the west exit in the new room to room 6709.

Keyword: These are the exit keywords, if any. Each keyword is separated by a space. These keywords are used with doors, so if you set a keyword of "gate", someone could open the door by typing "open gate".

Flags: These are flags that apply to the exit, such as secret, blocked, and no-bash.

Key: If this exit is lockable, this is the object that is used for the key, if any.

You can click the "Find" button to select the key object from a list.

Description: This is the description shown when someone looks at the exit using one of the keywords.

"Apply" stores the exit changes, "Cancel" closes the window without saving the exit changes, and "Delete" removes an existing exit.

Note: If you created a new target room for this exit using the "New" button, clicking "Cancel" or "Delete" will not delete it, nor will it delete the exit pointing from that new room to the one currently being edited.
