Editing Rooms

Back to the index

Rooms are what makes a zone take its shape -- they're the locations where players travel while experiencing the content of the zone.

The drop down list and left/right arrows let you navigate through the available rooms.

Name: This is the title of the room. It is also shown when someone uses the 'exits' command.

Description: This is the detailed description of the room.

Index #: This is the number of the room. Rooms should be numbered automatically when you create them.

Terrain: This is the terrain type of the room.

Flags: These are the various flags and settings that can be applied to a room, such as no-magic or single-file.

Current and Current Dir: If an aquatic room, this is the current strength in percent and the direction of the current.

Fall chance: If this room has a down exit, this is the chance of falling per turn.

Map sector: This is only used for worldmap zones.

The Extended Descriptions list box lets you add custom descriptions that are displayed when a player looks at specific keywords.

The exits section lets you add exits to other rooms. Click on the directional button to edit exit data for that direction. Buttons with grayed text do not have exits in that direction yet, while buttons with black text do.
