Editing Mobiles

Back to the index

Mobiles are the creatures of the game world. They can be friendly or hostile and can take the shape of anything from a Human to a dragon to an animated piece of furniture.

Alignment is a number from -1000 to 1000, with lower numbers being more evil-aligned and higher more good-aligned.

Name keywords is a list of words, seperated by spaces, which can be used to refer to the mob in commands like "consider", "kill", or "look".

The short description is the text that is displayed when the mobile is in combat.

The description is the text that is displayed when the mob is seen in a room.

The full description is shown when the mob is looked at.

The action flags and affect flags define the behaviors and abilities of the mob.
