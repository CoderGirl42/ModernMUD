Editing Objects

Back to the index

Objects are the things that players interact with in the game world. They can be weapons, armor, treasure, trinkets, trash, furniture, traps, or even food.

The index number is a unique number that is used by the mud to refer to the object.

The name keywords is a list of words, separated by spaces, that can be used to refer to the object when using commands like "get", "wear", "drop", and "wield".

The short description is an object's "short name", which is shown in a person's equipment list, inventory, and in othe rplaces.

The full description is the text that is shown when the item is sitting in a room or when it is examined.

Extended descriptions are descriptions or bits of text that are shown when someone looks or examines a specific keyword. For instance, for a "dagger with a ruby in the hilt" you could add an extra description that is shown on the keyword "ruby". When a person types "examine ruby", they will be shown a description of the ruby. An object can have any number of extra descriptions.

The "edit values" button lets you edit values specific to the type of item you are editing. For instance, weapons have a damage type (i.e. slashing) and a weapon type (i.e. scimitar or halberd) that can be set.

The "edit affect" button lets you add affects to an object.

The "edit spells" button lets you edit spells associated with an object. It is used for setting the spell affects of a potion, scroll, wand, or other magical item.

The "edit custom actions" button lets you add custom scripts to the object, such as procs, spells cast on command, or other special actions.
