ModernMUD Editor Overview

Back to the index

In order to edit an area (also known as a zone), you need to start by loading or creating one. This can be done via the File --> New or File --> Open menu commands.

Areas consist of a few major types of things:

Mobs
Objects
Rooms
Shops
Quests
Resets
Repop Points

Each of these can be edited using the icons on the left side the menu bar, in order. After the edit icons, the next six icons (directional arrows plus U and D) shift the map display north, south, west, east, up, or down. This lets you view parts of the map that hang off the edge of the display, or that are on different levels. The indicators after the icons show your X and Y offset (how far the map is shifted from center) as well as which level you are viewing.

The main window has two different views -- the map view (shown above), and the walkthrough view (shown below). The view can be changed by clicking the corresponding tab.Along the bottom of the window some information is displayed: The area name, the file location, and the quantity of each type of data in the area.

In walkthrough mode, commands can be entered as if you were inside the zone. Movement and information commands are available as are some editing commands. Type "commands" to see the full list of available commands. Walkthrough mode may be easier for some people to use when editing an area.

On the "tools" menu are two selections: "Renumber Area", which lets you automatically change all of the index numbers of objects, rooms, and mobs to start with a new number, and "Check Area", which analyzes the area and reports any items that haven't been filled in, such as room descriptions, mob names, and object keywords. Items in the error report can be double-clicked to jump to the edit window for that item.
